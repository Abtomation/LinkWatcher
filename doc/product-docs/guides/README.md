# Development Guides

This directory contains step-by-step guides and tutorials for developers working on the BreakoutBuddies project. These guides provide detailed instructions on how to perform specific development tasks.

## Available Guides

- [Testing Guide](guides/testing-guide.md) - Comprehensive guide to testing the BreakoutBuddies application, including setting up a local testing environment, running different types of tests, and troubleshooting common issues.
- [Test Environment Guide](guides/test-environment-guide.md) - Instructions for setting up and using the test environment for the Breakout Buddies app.
- [Supabase Local Setup](guides/supabase-local-setup-guide.md) - Guide for setting up Supabase locally for development.

## Purpose of Development Guides

Development guides serve several important purposes:

1. **Standardizing processes** - Ensures all developers follow the same procedures
2. **Knowledge sharing** - Captures and shares knowledge across the team
3. **Reference material** - Provides a reference for infrequently performed tasks

## Contributing to Guides

If you'd like to add a new guide or improve an existing one:

1. Follow the [Documentation Guide](/doc/process-framework/guides/guides/documentation-guide.md)
2. Use the [Guide Template](/doc/process-framework/templates/templates/guide-template.md) as a starting point
3. Include clear, step-by-step instructions with examples
4. Add screenshots or diagrams where appropriate
5. Include troubleshooting sections for common issues
