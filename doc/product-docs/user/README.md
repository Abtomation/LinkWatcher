# User Documentation

This directory contains documentation intended for end-users of the BreakoutBuddies application - a comprehensive platform for discovering, booking, and experiencing escape rooms, and sharing experiences with others.

## Directory Structure

- **handbooks/** - Step-by-step instructions for using the application
  - Account creation and profile management
  - Searching and filtering escape rooms
  - Booking process handbooks
  - Writing and submitting reviews
  - Connecting with friends
  - Planning group bookings

- **faq/** - Frequently asked questions
  - Account management
  - Booking process
  - Payment and cancellation policies
  - Social features
  - Provider portal usage
  - Technical troubleshooting

- **features/** - Detailed descriptions of application features
  - Search and discovery system
  - Booking and reservation system
  - Review and rating system
  - Social features and friend connections
  - Achievement and gamification system
  - Provider portal features

## Creating User Documentation

When creating user documentation:

1. Choose the appropriate subdirectory based on the content type
2. Use clear, non-technical language accessible to all users
3. Include screenshots and visual aids where helpful
4. Follow the [Process: Documentation Guide](../../process-framework/guides/guides/documentation-guide.md) for general documentation standards
5. Add your document to the [Process: Documentation Map](../../process-framework/documentation-map.md)

## Best Practices

- Write from the user's perspective
- Use simple, clear language
- Include plenty of screenshots and visual aids
- Provide step-by-step instructions for complex tasks
- Anticipate common questions and problems
- Test instructions by following them exactly as written
- Update documentation when features change
