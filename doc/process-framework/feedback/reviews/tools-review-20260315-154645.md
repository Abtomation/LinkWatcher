## Cross-Group Themes

### Theme 1: Structure Change task lacks delegation model (2/13 forms — but highest impact)
PF-TSK-014 absorbs specialized task processes instead of delegating. Caused a Phase 3 rollback when task creation bypassed PF-TSK-001 quality gates. Human feedback strongly advocates for orchestrator redesign. Appears in PF-TSK-014 group only but has framework-wide implications.

### Theme 2: Update script module import paths broken (4/13 forms)
Update-BugStatus.ps1 confirmed broken in both PF-TSK-041 and PF-TSK-007 groups. Prevented automation and required manual bug tracking updates. Other update scripts likely have the same issue.

### Theme 3: Post-fix checkpoint missing (1/13 forms — explicit human request)
Human explicitly requested a verification checkpoint after fix implementation and before closing bugs. Currently the AI agent can move straight from "tests pass" to closing without human sign-off.

### Theme 4: Automation scripts consistently strong (all groups)
New-ProcessImprovement.ps1, New-EnhancementState.ps1, New-ReviewSummary.ps1 all score 4.5-5.0. The automation layer is the strongest part of the framework.
