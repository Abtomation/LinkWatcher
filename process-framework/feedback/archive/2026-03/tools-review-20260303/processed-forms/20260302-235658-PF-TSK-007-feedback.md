- Automation scripts (Update-BugStatus.ps1) eliminated manual editing errors

### What could be improved
- The initial test design was too permissive (substring matching). A guideline about strong/negative assertions in bug regression tests would help.

### Specific suggestions
- Add a note to the bug fixing task about writing negative assertions (assert old value NOT present) in addition to positive assertions.
